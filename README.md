# My First Web Project | Landing Page

Проект представляет собой адаптивную веб-страницу, созданную в рамках курса ИКТ (14-я неделя СРО). [cite_start]Это моя первая работа, на которой я отработал(а) навыки верстки и деплоя. [cite: 1, 16]


## 🛠 Стек технологий
* **HTML5:** Семантическая верстка.
* **CSS3:** Работа со стилями, позиционирование элементов.
* **GitHub:** Управление версиями и хостинг проекта.

## 📋 Реализованный функционал
В рамках данного проекта было выполнено:
1. Создание структуры документа согласно стандартам HTML5.
2. Стилизация элементов и настройка внешнего вида через внешний файл `style.css`.
3. Публикация сайта в открытый доступ через GitHub Pages.
4. Оформление документации проекта.
